# Cloud Workspace Manager

## Overview

Cloud Workspace Manager is a full-stack portfolio project designed to demonstrate practical software engineering, backend development, Linux administration, DevOps, and cloud infrastructure skills.

The project is being developed incrementally, following a real-world engineering workflow.

---

## Technology Stack

### Frontend

- React
- TypeScript
- Vite

### Backend

- Node.js
- Express
- TypeScript

### Database

- PostgreSQL

### Infrastructure

- Ubuntu Server
- Docker
- Docker Compose
- Nginx

### Monitoring

- Prometheus
- Grafana

### Automation

- Python
- Bash
- GitHub Actions

---

## Development Environment

Development is performed on a dedicated Ubuntu Server virtual machine using:

- VS Code Remote SSH
- Git
- Ubuntu Server 24.04 LTS
- KVM / libvirt virtualization
